# LIFTSAFE
LIFTSAFE is a simple, cost-effective and easy to implemet lift safety mechanism.
A device which can ensure safety of an individual in an elevator during the event of a power cut ,by alarming the necessary authorities specifically based on human presence.LIFTSAFE offers a much more safer and reliable experience while using a lift,with features that include human presence detection and personalized warning through a text message and an alarm,LIFTSAFE ensures an individual’s complete security in case of disconnection or interruption of electricity.LIFTSAFE eliminates all the dependencies faced while commuting via elevator.Specific and dedicated safety mechanism ensures an individual’s safety and also reduces the rescue time.LIFTSAFE is only engaged if human presenceis detected, hence avoiding unnecessary alarms/warnings.
